TaskFlow ✅

TaskFlow is a clean, efficient, and fully functional Android application designed to help users manage their daily tasks seamlessly. Built natively in Android Studio, it features a modern UI and utilizes a local database for reliable data persistence.

Features

* Task Management (CRUD): Users can easily create, read, update, and delete tasks.
* Local Database: All tasks are securely saved on the device using Room, ensuring tasks are never lost when the app is closed.
* Modern UI/UX: Features a custom-designed, adaptive vector icon (blue/purple gradient) and an intuitive, user-friendly interface.
* Offline Functionality: Completely functional without an internet connection.

Built With

* IDE: Android Studio
* Language: Kotlin
* Database: Room Database
* UI: XML (Activities/Fragments, Adaptive Vector Drawables)
